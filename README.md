# Personal Portfolio Website

## Overview
This project is a responsive personal portfolio website built using HTML and CSS. It showcases personal information, technical skills, services, projects, and contact details through a clean, modern, and user-friendly interface.

## Tools Used
- HTML5
- CSS3
- Visual Studio Code
- GitHub

## Project Files
- index.html
- style.css

## Features
- Responsive website layout
- Fixed navigation bar with smooth scrolling
- Home section with introduction
- Services section
- Skills section
- Projects section with GitHub links
- Contact section
- Social media links (GitHub, LinkedIn, Email)
- Modern dark theme with interactive hover effects

## Results
The website successfully:

- Presents personal and academic information.
- Displays technical skills and services.
- Showcases completed projects with direct GitHub links.
- Provides responsive design for desktop and mobile devices.
- Offers a professional online portfolio for future employers and recruiters.
